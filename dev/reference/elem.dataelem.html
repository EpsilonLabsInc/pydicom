Data Elements (pydicom.dataelem)¶

Representation of DICOM data elements

DataElement(tag, VR, value[, ...])

Contain and manipulate a DICOM Element.

DataElement_from_raw(raw_data_element[, ...])

Return a DataElement created from raw_data_element.

RawDataElement(tag, VR, length, value, ...)

Container for the data from a raw (mostly) undecoded element.
